# VANTA User Research Summary — MetaLab Insights (March 12, 2026)

Synthesis from MetaLab user interviews.

---

## Top Features (Most Resonant)

1. **Smart Note** → Filtered Task List (#1 List View)
2. **Smart Contact List**
3. **Accelerators** (Simple Actions / Formulas)

## Connected Sources (Primary MVP)

- Calendar(s)
- Texts
- Emails
- Calls

## Least Resonant Features (Cutting Room Floor)

- Focused Brief
- Accelerator Library\*

## Curation + Control

- **Connected Sources**: Ability to control which sources are connected/disconnected.
- **Task List**: Ability to add, remove, organize task widgets; curating their list view.
- **Priorities**: Ability to set priority types (time, money, urgency), adjust priority levels (manual/auto); training AI to learn preferences and make smarter suggestions.
- **Modes**: Ability to curate and set modes based on different flows/moods of working (creative, executive, admin, DND); which sources, contacts, alerts, projects, list views are active.
- **\*Optional Screen Views**: Ability to select from the core list (1-3) views based on preferred personal workflow / style; making the list views optional vs. locking one global list interface.

---

## Task List (List View #1)

- **Priority Types (Categories)**: Being able to set and switch between stream of tasks based on priority type.
- **Distinct Priority Coding**: Assigning distinct color codes or clear markers that signal the level of task priority.
- **Added Useful Context**: Providing contextual snapshot for tasks; emails, calls, text prep.
- **Suggested Smart Actions (In-Task)**: Pulling up calendar with suggested times for scheduling; with ability to make real-time edits manually or using AI.
- **Curated Flow + Focused Widgets**: Each task appearing as dedicated widgets that can be expanded to take actions; mirroring a social feed vs. a flooded feed with abundance of tasks and actions blended together.
- **Functions As Curated Brief**: This list view serves as the true real-time daily brief curated to reflect the most relevant and important tasks, events and comms based on users pre-set categories; Replacing the "Focused Brief" alternative design view.

---

## Smart Note

- **Optimizing Native Behavior**: Users already rely heavily on their notes app and constantly dump ideas, thoughts, tasks and reminders; Our smart note optimizes their current experience.
- **Being Omnipresent**: Keeping the note feature one-tap away allows users to capture real-time thoughts as they move between screens.
- **Quick Actions + Shortcuts**: Giving users ability to instantly edit, save, share, tag and/or add media.
- **Converting Context**: VANTA being able to collect and turn the users shared context into suggested tasks, reminders or actions.
- **All-Purpose Workspace**: The smart note becomes an always-on workspace that allows users to create from a blank screen, input files and context to build on things in any moment.

---

## Smart Contact List (Insights)

- **Collecting + Adding Context**: Collecting core / most relevant details about person from point of adding contact (when, where, key detail or fact etc.)
- **Contact Tagging + Grouping**: Ability to assign custom tags or labels to specific contacts; Also being able to group specific contacts together based on shared alignment.
- **Outreach Reminders**: Reminders to make contact ie check-ins, follow-ups, happy birthday messages; similar to SuperPhone.
- **Automated Engagement**: Creating pre-set communications that can be automated to auto-reply and/or stay engaged with key contacts.
- **Surfacing Relevant Context (History)**: Surfacing key details around comms history (topics discussed, files shared, media captured).
- **\*Comms Prep**: Providing snapshot of relevant points and recent comms history when user receives calls / texts.

## Recommendations

### 1. Modes as Pre-Set Screen Views + Templates
Giving users the option to move between 3 core screen views or modes, each presented the same signal types curated in different streams. Giving user the ability to control connected sources, organize widgets and edit / set their priorities. **The Short**: Instead of designing one universal screen view for the task list, provide 3 core options reflecting the different stream types.

### 2. Making Smart Note an Omnipresent Feature
Allowing the smart note a fluid feature that's accessible from any screen; the hamburger button in top right that users can tap to expand for real-time thought sharing. The note collects context and suggests smart actions, in addition to having quick actions native to the note screen (share, tag, add etc.)

### 3. Limit Accelerators to Simple Formulas (Actions) Attached to Tasks
Instead of creating a library of formulas, position them as actionable extensions of user signals or tasks users discover in context of their workflow; whether suggested, searched or shared. Things like invoicing, drafting emails, creating a one-pager. This makes them a smart add-on vs a highlighted feature/focal point.

### 4. Explore Smart Contact Interface (UX)
We have an opportunity to design the new contact card experience for this massive cohort; from collecting context, surfacing history and pinning key details to automating comms and setting reminders.

## Validated Points

| Principle | Insight |
|---|---|
| **Control** | Users want ability to control connected sources, priorities, actions, and network access |
| **Curation** | Users deeply value curating their workflow — categories, priorities, communications, flows, and information sources |
| **Personal Ops** | Need for smart personal operations tool far exceeds need for creative productivity optimizer |
| **The Mood Is The Mode** | Different users have different modes of working; premium on simple optionality for screen views, modes, and controls |
| **Less Content, More Context** | Hyper-focused, curated feeds preferred; less pressure, more preparation and clarity for decisions |
| **Streamlining Is Success** | One clear, connected, curated stream of important tasks is an undeniable need that doesn't exist as a singular service |
